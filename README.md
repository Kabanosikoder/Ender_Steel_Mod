# Ender Steel Mod

A Minecraft mod that adds Ender-infused steel equipment with unique abilities.

## Features

### Ender Steel Armor Set
- **Full Set Bonus: Evasion**
  - Dodge up to 5 hits
  - 2-minute cooldown after all charges are used
  - Teleports you to safety when dodging
  - Visual and audio feedback for dodges

### Ender Steel Sword
- **Pearl Storage System**
  - Store up to 10 ender pearls
  - Right-click with an ender pearl to store it
  - Shows current pearl count in tooltip
- **Combat Effects**
  - 10% base chance to teleport enemies on hit
  - Deals 2.5 bonus magic damage when teleporting
  - Chance increases by 2.5% with each successful teleport
  - Streak resets when out of pearls

### Ender Steel Scythe
- **Gazing Void Enchantment**
  - Very rare enchantment exclusive to the Ender Steel Scythe
  - Right-click to activate (10-second cooldown)
  - Next hit:
    - Blinds and immobilizes target
    - Displays an ominous ender eye overlay
    - Plays enderman scream sound effect

### Tools and Equipment
- Ender Steel Pickaxe
- Ender Steel Axe
- Ender Steel Shovel
- Ender Steel Block

### Crafting Materials
- Ender Scrap
- Ender Steel Ingot

## Requirements
- Minecraft 1.20.1
- Fabric Loader
- Fabric API

## Installation
1. Install Fabric Loader and Fabric API
2. Place the mod JAR file in your mods folder
3. Launch Minecraft and enjoy!

## Recipes
All items can be found in the Ingredients creative tab.
